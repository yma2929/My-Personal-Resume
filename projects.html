Introduction to Database Project

Project Title:Resort Database Managment System
Project Description:This project involved designing and implementing a database management system for a resort. The project included creating tables, defining relationships, and writing queries to retrieve and update data. The goal of the project was to provide a user-friendly interface for managing resort operations and improving efficiency.
