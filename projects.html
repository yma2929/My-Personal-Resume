Digital Logic Design Project

Project Title:ALU - Arthimetic Logic Unit
Project Description: This project involves designing and simulating a basic Arithmetic Logic Unit (ALU), a critical component of a CPU responsible for performing arithmetic and logical operations. The ALU in this project is designed to execute two primary operations: addition and subtraction. It uses two 4-bit inputs (A and B) and a mode selector (M0) to produce a 4-bit output (F), which is displayed on a 7-segment LED via a BCD-to-7-segment decoder. The project is designed to provide a hands-on learning experience with digital logic circuits, using basic components such as gates and switches to perform fundamental arithmetic operations.
